# بروتوكول التنفيذ الإلزامي
كل أمر بناء لا يُعتبر منجزًا حتى:
1. commit برسالة تحمل رقم الأمر (مثال: feat(04-B2): custodies module)
2. push إلى origin/main + لصق نطاق الـ hash
3. git status = up to date, working tree clean
4. php artisan test كاملة (لا فلتر) + لصق سطر الملخص
5. براهين grep المطلوبة في الأمر نصًا
6. قائمة الملفات المنشأة/المعدلة
