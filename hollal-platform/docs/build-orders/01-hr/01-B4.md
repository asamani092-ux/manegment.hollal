## Objective

Attendance program (per-employee activation): base hours by HR, check-in/out on dashboard, overtime dropdown locked/unlocked by manager, monthly days setting applied by scheduler.

## Spec reference

docs/specs/spec-01-hr.md §7 (extended attendance program)

## Deliverables

- Per-employee attendance activation
- Dashboard check-in/out (activates 00-B5 placeholder)
- Overtime dropdown manager lock/unlock
- Monthly days setting via scheduler

## Required grep proofs

- `check-in|check_in|check_out`
- `attendance`
- `overtime.*lock|locked`

## Tests

- Inactive employees cannot check in
- Manager unlock required for overtime dropdown when locked
- Scheduler applies monthly days setting
