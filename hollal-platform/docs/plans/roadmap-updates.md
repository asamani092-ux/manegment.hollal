# roadmap-updates.md

تحديثات خارطة الطريق بعد إقرار نظام أوامر البناء (يوليو 2026).

## ما انتقل من «مؤجّل» إلى أوامر مرقّمة

| سابقًا في ROADMAP.md | الآن |
|----------------------|------|
| صناعة البرامج والمقررات | **06A-*** |
| المشتريات والعهد والأصول | **04-B3 / 04-B5** (عهد وأصول؛ المشتريات تبقى لاحقة إن لزم) |
| قياس الأثر | **06B-B4** + تقارير **08-B2** |
| إشعارات البريد/الجوال وبوابة الشركاء | **00-B3** + بوابة **05-B5** |
| مركز الموافقات الموحد | جزئيًا عبر طوابير الاعتماد في الوحدات؛ مركز موحّد يبقى لاحقًا إن لزم |
| البحث الشامل والإضافة السريعة | يبقى post-build ما لم يُدرج أمرًا |
| المتجر والمطبوعات | يبقى مؤجّلًا |

## الاستضافة
